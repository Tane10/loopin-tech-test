# loopin-tech-test

If you would like to run the service and get an output without running the tests then please **uncomment lines 81 - 88** before running using nodemon or in prod

## Install

``` bash
npm i 
```

## Run tests

``` bash
npm run test
```

## Build

``` bash
npm run build
```

## Run using nodemon

``` bash
npm run dev
```

## Run in prod

``` bash
npm run build && npm run start
```
